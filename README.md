# 💎 React Quiz App

A Simple React Quiz App 

## See it Live [HERE](https://quiz-app-jatin8898.herokuapp.com) 

### Version
1.0.0

## 📝 Usage

### Clone this Repo
```
git clone https://github.com/Jatin-8898/react-quiz-app.git
```
### Installation

```sh
$ npm install
```

### :ballot_box_with_check: Run

Then go to http://localhost:8000

```sh
$ npm start
```
